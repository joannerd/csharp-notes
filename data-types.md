# Data Types

- [Built-in Types]
- [void]: indicate absence of a type or that a method doesn't return a value

## Types

- Strings:
  - `string` keyword = alias for `String`
  - Helpful methods:
    - `.TrimStart()`
    - `.TrimEnd()`
    - `.Trim()`
    - `.Replace("a", "b")`
    - `.ToUpper()`
    - `.ToLower()`
    - `.Contains()`
    - `.StartsWith()`
    - `.EndsWith()`
    - `new StringBuilder()` + `.Append()`
- Numbers:
  - int `1`
  - double `1.0`
  - decimal `1.0M`
  - [Math.PI] `public const double PI = 3.1415926535897931;`
